# staff-app
Aplikasi pengaturan data staff. Akan berintegrasi dengan inventory-app yang mengatur inventory/barang yang bisa di-assign kepada seorang staff.

## Getting started
Untuk mempermudah, pada root saya sediakan sql `DatabaseTugas24.sql` untuk langsung dijalankan agar bisa langsung create database yg dibutuhkan untuk run program ini.

## Testing
Saya juga menyediakan hasil export endpoints collections dari Postman yang bisa langsung anda import di Postman anda.
Pada file dengan nama `StaffApp.postman_collection.json`.
